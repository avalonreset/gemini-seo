# Codex Notes

This repository uses standard `SKILL.md` files under `skills/`.

Install by copying the `skills/*` directories into `$CODEX_HOME/skills` or by
running `install.sh` / `install.ps1`. The main entrypoint is `skills/seo/SKILL.md`;
the root `SKILL.md` is a compatibility wrapper.

For audits, read `skills/seo/SKILL.md`, then load the specific sub-skill named
by the requested `/seo` command.
