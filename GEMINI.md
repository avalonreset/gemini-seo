# Gemini Notes

The release asset `gemini-seo.skill` packages the root `SKILL.md`, the
`skills/` tree, scripts, schema templates, and docs.

Install the packaged skill with:

```bash
gemini skills install path/to/gemini-seo.skill --scope user
```

After installation, use natural language SEO requests or `/seo ...` commands.
The main workflow router lives at `skills/seo/SKILL.md`.
